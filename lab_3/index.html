Описание методов:

Глобальная пороговая (среднее): Использует среднее значение яркости всего изображения как порог.
Глобальная пороговая (Оцу): Автоматически находит оптимальный порог, максимизируя межклассовую дисперсию.
Локальная пороговая (среднее): Использует среднее значение в локальном окне.
Локальная пороговая (медиана): Использует медиану в локальном окне.
Адаптивная пороговая: Адаптируется к локальным изменениям освещения.
Параметры:
• Размер окна: размер локального окна для локальных и адаптивных методов (3-51, нечетное)
• Порог C: константа, вычитаемая из порога для локальных методов (1-50)
